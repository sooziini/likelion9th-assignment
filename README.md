## 과제 제출을 위한 최초 설정
**1. 팀장 레포 fork**   
   : 자신의 레포에 fork 뜬 레포 자동 생성   
   
**2. `git clone [fork 뜬 자신의 레포 주소]`**   
   : fork 뜬 자신의 레포 파일을 로컬에 생성하는 명령   
   
**3. `git remote add upstream [팀장 레포 주소]`**   
   : 과제 시작 전 팀장 레포의 업데이트된 파일들을 받아오기 위해 팀장 레포를 연결하는 명령
